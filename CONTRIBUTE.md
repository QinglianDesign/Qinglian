#### 开发准备

- Node.js
  `version >= 14.x`
  推荐安装方式：

  1. nvm
     [nvm 安装方式](https://github.com/nvm-sh/nvm)
  2. volta
     [volta 安装方式](https://github.com/volta-cli/volta)

- Rust
  `安装最新版本即可`
  [Rust 安装方法](https://www.rust-lang.org/tools/install)

#### 常见命令

- 单元测试 `npm run test`
- 新增子包 `npm run init <package-name>`
- 构建 `npm run build`
